# Notes App

A simple CLI notes manager in Python. Built as part of an assignment comparing
**vibe coding** vs **spec-driven development** (OpenSpec).

## Run
```bash
python3 app.py
```

## Branches
- `main` — base skeleton
- `vibe-coding` — search feature added via raw LLM prompting
- `openspec` — search feature added via OpenSpec spec-driven workflow

See `DIFFERENCES.md` on the `openspec` branch for the comparison.
